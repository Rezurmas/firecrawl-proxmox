## 🤝 Wkład / Contributing / Mitwirken

1. Skopiuj `README.en.md` jako bazę. / Copy `README.en.md` as a base. / Kopieren Sie `README.en.md` als Basis.
2. Przetłumacz cały tekst (NIE zmieniaj kodu, placeholderów ani ścieżek!). / Translate all text (DO NOT change code, placeholders, or paths!). / Übersetzen Sie den gesamten Text (Code, Platzhalter und Pfade NICHT ändern!).
3. Nazwij plik `README.XX.md` (gdzie XX to kod języka, np. `fr`, `es`, `ja`). / Name the file `README.XX.md` (where XX is the language code, e.g. `fr`, `es`, `ja`). / Benennen Sie die Datei `README.XX.md` (wobei XX der Sprachcode ist, z. B. `fr`, `es`, `ja`).
4. Dodaj wpis w tym pliku (`LANGUAGES.md`). / Add an entry in this file (`LANGUAGES.md`). / Fügen Sie einen Eintrag in dieser Datei (`LANGUAGES.md`) hinzu.
5. Wyślij Pull Request! / Submit a Pull Request! / Reichen Sie einen Pull Request ein!
